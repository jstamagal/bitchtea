# 🦍 BITCHTEA: GLOSSARY

The words of the Green Dark.

- **`APE`**: The agent. You. Strength through action.
- **`KING`**: The user. The alpha who gives the commands.
- **`Fantasy`**: The internal Go library (`charm.land/fantasy`) that shims multiple LLM providers.
- **`Bootstrap Context`**: Injected messages at the start of every session (Context files, MEMORY.md, Persona Anchor).
- **`Context Files`**: Markdown files (`AGENTS.md`, `CLAUDE.md`) discovered in the worktree to ground the agent.
- **`Durable Memory`**: Facts extracted during compaction and saved to daily log files for long-term recall.
- **`Green Dark`**: The philosophy of minimal friction, raw speed, and no corporate politeness.
- **`Hot Memory`**: The `MEMORY.md` file in the project root, used for immediate, high-priority context.
- **`IRC Context`**: Routing metadata that mimics IRC channels and queries to keep different tasks separated.
- **`Persona Anchor`**: A synthetic exchange at the end of the bootstrap that locks the agent into its APE persona.
- **`Steering`**: Typing a message while the agent is already working. It is queued and fires as soon as the agent finishes.
