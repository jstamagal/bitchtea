# 🦍 GLOSSARY 🦍

The words we use in the canopy.

- **Agent**: The brain loop that thinks and uses tools.
- **Bootstrap Context**: The first messages (like `CLAUDE.md`) given to the agent to teach it project rules.
- **Channel**: A workspace context (like `#refactor`). Each channel can have its own `HOT.md` memory.
- **Fantasy**: The autonomous state where the agent runs tools and continues without waiting for you.
- **Follow-up**: An automatic "next step" or "idea" generated by the agent after a tool finishes.
- **Hyper**: High-speed tool execution mode.
- **Persona Anchor**: Fixed messages that define the agent's name and core personality.
- **Profile**: A saved bundle of settings (API key, model, provider, base URL).
- **Scope**: The boundary of memory. Can be `Root` (global) or `Channel` (local).
- **Transcript**: The visual record of your conversation in the viewport.
